ALGORITMO PARA LAVARSE LAS MANOS

  • Abrir el grifo.
  • Mojarse las manos con agua.
  • Aplicar jabón en las manos.
  • Frotar las manos para crear espuma.
  • Enjuagar las manos con agua.
  • Cerrar el grifo.
  • Secarse las manos con una toalla.

ALGORITMO PARA ENCONTRAR EL NÚMERO MÁS GRANDE EN UNA LISTA

  • Comienza el algoritmo.
  • Empieza con el primer número de la lista como el más grande.
  • Mira cada número de la lista uno por uno.
  • Si el número es más grande que el actual máximo, cámbialo por ese número.
  • Repite esto con todos los números.
  • Cuando termines, el máximo será el número más grande de la lista.
  • Termina el algoritmo.
